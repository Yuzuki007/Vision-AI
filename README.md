Here's VisionAI — your serverless AI object detection

🧠 How It Works
Uses TensorFlow.js + COCO-SSD model — the same technology behind AWS Rekognition, running 100% in your browser. Detects 80 object classes including people, cars, animals, phones, food, furniture, and more.

📱 Features
🔍 Detect Tab

Upload from photo library or take a photo with your camera
Bounding boxes drawn with glow effects and corner brackets
Each object gets a unique color + emoji label
Toggle boxes, labels, and zoom mode from the toolbar
Tap any detection card to flash-highlight it on the image

🎥 Live Tab

Real-time detection on your camera feed
Updates continuously frame by frame
Shows top 5 detected objects live

🕓 History Tab

Save annotated images with detections
Thumbnail previews with object chips
Tap to reload any past session

⚙️ Settings

Confidence threshold (30%–80%)
Max detections per image
Colored vs single-color boxes
Box thickness control
Show/hide confidence scores

📤 Export Options

Save annotated image (with boxes drawn)
Export as JSON (bbox coordinates + confidence)
Export as CSV
